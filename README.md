# AI-Driven Sustainability Report Intelligence Platform

An enterprise-grade platform for automating ESG reporting using AI, RAG, and multi-framework intelligence.

## 🚀 Tech Stack
- **Frontend**: Next.js 14, TailwindCSS, Framer Motion, Chart.js, Lucide Icons.
- **Backend**: FastAPI (Python), OpenAI GPT-4, Pandas, SQLAlchemy.
- **AI**: Retrieval-Augmented Generation (RAG) for GRI, SASB, and UN SDG alignment.

## 📂 Project Structure
- `/frontend`: Next.js web application.
- `/backend`: FastAPI intelligence engine.
- `/data`: Storage for ingested documents and framework logic.

## ⚡ Quick Start

### 1. Backend Setup
1. `cd backend`
2. `pip install -r requirements.txt`
3. Add your `OPENAI_API_KEY` to `.env` (already pre-configured with the key you provided)
4. `python main.py`

### 2. Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
